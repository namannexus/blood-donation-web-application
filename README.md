# Blood Donation Website (clean)
- Next.js 14 + TailwindCSS v3.4.13
- No NextAuth (simple admin password via env)
- MongoDB (Mongoose) for DB
- Nodemailer helper for optional email notifications

Quick start:
1. Copy `.env.local.example` -> `.env.local` and fill values.
2. `npm install`
3. `npm run dev`
4. Open http://localhost:3000
